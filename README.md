# Site de Agendamento — Tiro Olímpico

Website moderno em **Next.js** para agendamento de treinos de tiro olímpico, usando a logo fornecida em `public/logo.png`.

## Stack utilizada

- Next.js
- React
- Tailwind CSS
- GSAP
- Lenis
- Three.js
- React Three Fiber
- Framer Motion

## Funcionalidades

- Landing page moderna e responsiva
- Animações com Framer Motion e GSAP
- Scroll suave com Lenis
- Cena 3D decorativa com Three.js / React Three Fiber
- Formulário de agendamento de treinos
- Formulário de contactos
- Painel administrativo com login
- Listagem de agendamentos e contactos
- Exportação CSV de agendamentos
- Eliminar registos no painel
- Persistência local via `localStorage`

## Login do painel admin

- Utilizador: `admin`
- Palavra-passe: `tiro2026`

> Nota: esta versão guarda os dados no navegador para demonstração. Para produção, deve ser ligada a uma API/base de dados, autenticação real e notificações por email/SMS.

## Como executar

```bash
npm install
npm run dev
```

Depois abra:

```text
http://localhost:3000
```
